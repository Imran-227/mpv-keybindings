

This configuration file try to match the vlc player hotkeys behavior !

[Installation]

**On windows**
Copie the "input.conf" to this location: 
	"C:\users\USERNAME\AppData\Roaming\mpv\input.conf"
If it does not  exit create it.

**Linux / Mac**
Copie the "input.conf" to this location :
	"/usr/local/etc/mpv/input.conf"
